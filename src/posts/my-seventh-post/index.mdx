# Programming
So, coding - it's a rather interesting subject since you can make anything with the right techniques and imagination. Well, I'll show you some easy code.

## HTML

```html
<body>
  <main>
    <article>
      <h1>hello world!</h1>
      <p>We shall say hello to the world!</p>
    </article>
  </main>
</body>
```

## CSS
```css
  *{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }

  body {
    background-color: #007fff;
  }

  main {
    width: 50%;
    background-color: white;
  }

  article{
    width: 100%;
  }

  h1 {
    color: black;
    font-size: 3rem;
    font-weight: 600;
  }
```

Well, what's written above won't make it look nice unless you centralize the content and change more things, but I'm far too lazy to do that right now.

| Syntax    | Description | Type         | code         |

| Header    | Title       | HTML Element | h1  and up   |

| Div       | Section     | HTML Element | div          |
